SafeTalk - Mental Health Support
SafeTalk is a digital sanctuary designed to provide a safe space for mental health reflection, peer support, and emotional well-being. Built with the MERN stack, the platform enables users to track their moods, share personal experiences, and connect with a supportive community.

🚀 Live Demo
Access the platform here: https://safetalk-project-git-main-harshita-verma12s-projects.vercel.app/

✨ Key Features
Mood Sanctuary: An interactive, color-coded mood matrix that helps users identify and understand their emotional states with psychological context.

Community Circle: A social-style feed where users can share their journeys, support one another, and engage in meaningful discussions within a safe environment.

Daily Reflections: A private journaling tool for documenting thoughts, feelings, and progress toward mental wellness.

Responsive Design: A clean, elegant, and professional user interface optimized for both desktop and mobile devices.

🛠️ Built With
Frontend: React.js, Vite, Tailwind CSS, Axios.

Backend: Node.js, Express.js.

Database: MongoDB Atlas.

Deployment: Vercel (Frontend), Render (Backend).

💡 Philosophy
SafeTalk is built on the belief that vulnerability is a strength. Our platform aims to reduce the stigma surrounding mental health by providing tools for self-awareness and community empathy.
