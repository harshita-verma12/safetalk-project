# SafeTalk

Mental health support web app (MERN) scaffold.

## Setup

1. Backend:
   - `cd backend`
   - `npm install`
   - create `.env` with `MONGO_URI` and `JWT_SECRET`
   - `npm run dev`
2. Frontend:
   - `cd frontend`
   - `npm install`
   - `npm run start`

## Endpoints
- /api/auth/register
- /api/auth/login
- /api/community
- /api/profile/journals
- /api/ai/assistant

## Features
- Login/Register
- Community stories
- Journaling + moodboard
- Blogs + static posts
- AI assistant
- Responsive UI
